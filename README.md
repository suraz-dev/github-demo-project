# GitHub Demo Project

A small demonstration repository for connecting GitHub with Codex and managing a project directly from a local workspace.

## Getting started

Clone the repository and open the project directory:

```bash
git clone https://github.com/suraz-dev/github-demo-project.git
cd github-demo-project
```

This repository is intentionally lightweight, so there are no dependencies or build steps required yet.

## Workflow

1. Create a focused change in a separate branch.
2. Review the diff locally.
3. Commit the change with a concise message.
4. Push the branch and open a pull request.

## Project status

This is a starter project. Future updates can add example code, automated tests, and GitHub Actions workflows.

## Contributing

Small, focused contributions are welcome. Please describe what changed and include any relevant verification steps in the pull request.
